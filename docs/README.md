# Documentation

This folder contains detailed documentation for the aqie-dc-backend service.

## CDP Uploader Integration

The `cdp-uploader/` folder contains all documentation related to the CDP Uploader file upload integration:

- **[CDP_UPLOADER_IMPLEMENTATION.md](./cdp-uploader/CDP_UPLOADER_IMPLEMENTATION.md)** - Complete implementation guide covering architecture, setup, and configuration
- **[CDP_UPLOADER_TESTING.md](./cdp-uploader/CDP_UPLOADER_TESTING.md)** - Quick reference with curl commands for testing the upload flow in dev/test environments
- **[TESTING_GUIDE.md](./cdp-uploader/TESTING_GUIDE.md)** - Comprehensive testing guide for local development with CDP Uploader

## Quick Start

For developers new to this project:

1. Read the main [README.md](../README.md) in the project root
2. For file upload functionality, start with [CDP_UPLOADER_TESTING.md](./cdp-uploader/CDP_UPLOADER_TESTING.md)
3. For in-depth implementation details, see [CDP_UPLOADER_IMPLEMENTATION.md](./cdp-uploader/CDP_UPLOADER_IMPLEMENTATION.md)
